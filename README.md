# BackendChallenge

Backend python CLI app offering CRUD functionality for a local database. Tracking Authors and related books, you can create, update and delete entries as well as view all books by specific authors. Also utilizes Open Library to query books by title and save the book and author to the database.


# SETUP
## -Open the project folder in VSCode

## -Run pip install -r requirments.txt in the terminal

## -Run main.py (F5)

## -Follow the Prompts in the Terminal
